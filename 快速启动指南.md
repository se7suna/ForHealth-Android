# For Health 快速启动指南

## 🚀 5分钟快速启动

### 1. 启动后端服务 (30秒)

```bash
# 进入后端目录
cd backend

# 启动服务
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

看到以下信息表示成功：
```
INFO:     Uvicorn running on http://0.0.0.0:8000
INFO:     Application startup complete
```

### 2. 验证后端 (30秒)

浏览器访问：http://localhost:8000/docs

应该能看到Swagger API文档界面。

### 3. 启动前端 (2分钟)

在Android Studio中：
1. 打开项目
2. 等待Gradle同步完成
3. 点击运行按钮 ▶️

### 4. 测试应用 (2分钟)

#### 注册新用户：
1. 点击"注册"
2. 填写信息：
   - 用户名：`测试用户`
   - 邮箱：`test@test.com`
   - 密码：`123456`
   - 确认密码：`123456`
3. 点击"注册"

#### 填写身体数据：
1. 性别：选择性别
2. 出生日期：选择日期
3. 身高：175cm
4. 体重：70kg
5. 活动水平：中度活跃
6. 健康目标：减重，目标65kg，10周

#### 查看个人信息：
1. 在主页点击右上角图标
2. 查看完整的个人信息

## ✅ 成功标志

- 后端日志显示API请求
- 前端显示"保存成功"
- 个人信息页面正确显示中文数据

## 🔧 常见问题

### 后端启动失败
```bash
# 检查MongoDB是否运行
# 检查Python环境
# 检查依赖是否安装
pip install -r requirements.txt
```

### 前端连接失败
- 检查BASE_URL配置（默认：`http://10.0.2.2:8000/`）
- 模拟器使用`10.0.2.2`
- 真机使用电脑局域网IP

### Gradle同步失败
```bash
# 清理项目
./gradlew clean

# 重新构建
./gradlew build
```
