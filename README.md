# for_health

## Name
就要健康-卡路里消耗记录系统

## Description
一个记录食物卡路里摄入（收入）和运动卡路里消耗（支出）的类记账系统，为扩展功能加入了AI识别与健康助手。

## 开发注意事项
1.做任何功能前请开一个分支，在分支上完成任务后提交Merge Request，在描述中写出相关issue和所做的工作

2.git commit请符合commit message规范，简要描述工作

3.doc文件夹中存放了前后端的说明文档，开发前请仔细阅读。（requirements.txt后端依赖在backend中）

4.目前不同分支的gitignore不同，忽略的文件不一致，在切换分支时可能产生bug
